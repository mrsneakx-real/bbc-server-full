# Bbc-Server-Full
BBC Server Full Version

On this repo you will find all the World downloads and so mutch more. This repo and the OneDrive folder fall under the EULA you can find in this repo. (EULA.md)

If you would like to download and play this world on your Pc start by downloading the folder on the Bbc Sharepoint site. Make sure to read the README there also!

After that unzip the file into your Downloads folder. Then locate your Minecraft world folder by pressing Windows + R and typing: %APPDATA%\.minecraft\saves into the input field. 
Once that step is complete open a new Explorer window and navigate to your Downlads folder.
Then locate your save-server folder you earlyer unziped. (.\Downlads\save-server)
Now move the save-server folder into your Minecraft saves folder. (.\AppData\.minecraft\saves)
As a last step start resp. restart Minecraft. You should now find the World in your Singleplayer world selection.

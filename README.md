# Bbc-Server-Full
BBC Server Full Version

On this Repo you will find all the World downloads and so mutch more.

If you like to download and play this world on your Pc start by downloading the Tag with the name "release".
After that unzip the file into your Downloads folder. Then locate your Minecraft world folder by pressing Windows + R and typing: %APPDATA%\.minecraft\saves into the input field. Once that step is complete open a new Explorer window and navigate to your Downlads folder.
The locate your save-server folder you earlyer unziped. (.\Downlads\save-server)
Now move the save-server folder into your Minecraft saves folder. (.\AppData\.minecraft\saves)
As a last step start resp. restart Minecraft. You should now find the World in your Singleplayer world selection.
